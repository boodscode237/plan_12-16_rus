Модуль 3

Машинное обучение и NLP (14 недель)

Практический модуль, посвященный основным алгоритмам машинного обучения и обработке естественного языка. Используем Scikit-learn и другие библиотеки для создания реальных моделей.

Обучение с учителем
  • • Классификация
  • • Регрессия
  • • Примеры: спам-фильтры, прогнозирование

Мастер-класс: "Классификация ирисов"

Работаем с классическим набором данных Iris. Используем KNN для классификации цветков по измерениям. Практикуем загрузку данных, разделение на выборки, обучение модели и оценку точности.

Недели 19-20: Регрессия (линейная, полиномиальная)

Ключевые концепции:

Линейная регрессия

y = wx + b

  • • Метод наименьших квадратов
  • • Функция потерь MSE
  • • Интерпретация коэффициентов
Полиномиальная регрессия

y = w₀ + w₁x + w₂x² + ...

  • • Моделирование нелинейных зависимостей
  • • Проблема переобучения
  • • Выбор степени полинома

Недели 21-22: Классификация (KNN, логистическая регрессия, SVM)

K-ближайших соседей
  • • Принцип "голосования"
  • • Выбор k и метрики расстояния
  • • Плюсы: простота, интерпретируемость
  • • Минусы: медленный на больших данных
Логистическая регрессия
  • • Сигмоидная функция
  • • Вероятность класса
  • • Максимальное правдоподобие
  • • Интерпретация коэффициентов
Метод опорных векторов
  • • Разделяющая гиперплоскость
  • • Максимизация зазора
  • • Ядерный трюк
  • • Линейное и RBF ядра

Мастер-класс: "Спам-фильтр"

Создаем фильтр спама с использованием различных алгоритмов классификации. Работаем с реальными данными писем, преобразовываем текст в числовые признаки (Bag of Words) и сравниваем эффективность моделей.

Недели 23-24: Кластеризация (K-средних)

Алгоритм K-средних:

Шаги алгоритма:
  1. Инициализация K центроидов
  2. Назначение точек ближайшим центроидам
  3. Пересчет позиций центроидов
  4. Повторение до сходимости
Выбор K:
  • • Метод локтя (Elbow method)
  • • Анализ силуэта (Silhouette)
  • • Визуальная оценка

Мастер-класс: "Сегментация клиентов"

Применяем K-средних для сегментации клиентов интернет-магазина. Анализируем поведенческие данные, определяем оптимальное число кластеров и разрабатываем маркетинговые стратегии для каждого сегмента.

Недели 25-26: Нейронные сети и глубокое обучение

Основные компоненты:

  • • Нейрон: взвешенная сумма + функция активации
  • • Слои: входной, скрытые, выходной
  • • Функции активации: ReLU, сигмоидная, softmax
  • • Обучение: обратное распространение ошибки
  • • Оптимизация: градиентный спуск

Мастер-класс: "Распознавание рукописных цифр"

Создаем и обучаем нейронную сеть для классификации цифр из набора данных MNIST. Используем Keras/TensorFlow для реализации сети с одним скрытым слоем. Визуализируем процесс обучения и анализируем результаты.

Недели 27-28: NLP - основы, токенизация, стемминг

Предобработка текста
  • • Токенизация
  • • Нормализация
  • • Удаление стоп-слов
  • • Стемминг и лемматизация
Векторизация
  • • Мешок слов (Bag of Words)
  • • TF-IDF
  • • One-hot encoding
  • • Word embeddings (кратко)
Библиотеки
  • • NLTK
  • • spaCy
  • • Scikit-learn
  • • Gensim (кратко)

Мастер-класс: "Анализ тональности отзывов"

Разрабатываем систему анализа тональности текста. Обрабатываем отзывы о фильмах, преобразуем текст в числовые признаки с помощью TF-IDF и обучаем классификатор определять, является ли отзыв позитивным или негативным.
